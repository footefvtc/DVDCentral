# Purpose
Connect and use the Azure Open AI

# Steps
Configure Azure
1. Login to the Azure Portal https://portal.azure.com/
2. Search for OpenAI and click on the OpenAI resource
3. Click on Open AI Studio or Foundry
4. Get the API key and endpoint URL

Guide users through connecting a Console Application with Open AI
1.	UIConsole - Add additional code from my repo
2.  Add the API key and endpoint URL to the appsettings.json file
3.  Run the application and test the connection to Open AI
